# 投稿上传文件说明

本目录整理的是 Springer Nature 模板版本的投稿材料。投稿系统如果要求上传 LaTeX 源文件，请保持 `source/` 和 `figures/` 的相对目录结构不变。

## 建议上传清单

| 文件或目录 | 投稿系统文件类型建议 | 说明 |
| --- | --- | --- |
| `aej-sn-article.pdf` | Manuscript / Main document / PDF | 已编译的完整论文 PDF，用于投稿预览和审稿。 |
| `source/aej-sn-article.tex` | Manuscript source / LaTeX source | 主 LaTeX 源文件。 |
| `source/aej-refs.bib` | Bibliography / BibTeX file | 参考文献数据库。 |
| `source/sn-jnl.cls` | LaTeX class / Style file | Springer Nature 模板类文件，编译源文件需要。 |
| `source/sn-mathphys-num.bst` | BibTeX style / Style file | 当前稿件使用的 BibTeX 样式文件。 |
| `figures/*.pdf` | Figure / Artwork / Source file | 正文实际引用的图表文件，共 34 个。 |
| `title_running_head_abstract.txt` | Metadata reference / Not for review | 投稿系统填写 Title、Running head、Abstract 时可复制使用。 |

## 可选上传方式

如果投稿系统只允许上传一个 LaTeX 源码压缩包，可以上传 `latex_source_package.zip`。该压缩包包含：

- `source/aej-sn-article.tex`
- `source/aej-refs.bib`
- `source/sn-jnl.cls`
- `source/sn-mathphys-num.bst`
- `figures/` 目录下所有正文引用图表

## 不需要上传的文件

以下文件是本地编译中间产物，不需要上传，也未放入本投稿包：

- `.aux`
- `.log`
- `.out`
- `.bbl`
- `.blg`
- `.abs`

## 备注

当前论文主文件采用 Springer Nature 模板：

```latex
\documentclass[pdflatex,sn-mathphys-num]{sn-jnl}
```

主文件中的图表路径为：

```latex
\graphicspath{{../figures/}{figures/}}
```

因此从 `submission_upload/source/` 目录编译时，可以正常读取 `submission_upload/figures/` 下的图表。
